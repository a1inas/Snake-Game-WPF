# 🐍 Snake — классическая игра "Змейка" на WPF

**Версия:** 1.0.0  
**Платформа:** Windows  
**Технологии:** C#, WPF, .NET 6.0

## О проекте

**Snake** — это классическая аркадная игра, в которой игрок управляет змейкой, перемещающейся по игровому полю. Цель игры — съедать пищу, которая появляется на поле, и расти, избегая столкновений с собственным хвостом или стенами игрового поля.

Проект разработан в рамках курсовой работы и представляет собой полностью функциональное приложение с интуитивно понятным интерфейсом, звуковым сопровождением и возможностью игры как в одиночку, так и вдвоем.

## Геймплей

### Основная цель
Съедать пищу, появляющуюся на поле, увеличивая длину змейки и набирая очки, избегая столкновений со стенами и собственным телом.

### Игровой процесс
- Змейка постоянно движется в выбранном направлении
- С каждым съеденным кусочком пищи змейка увеличивается в длину
- При столкновении со стеной или собственным телом игра завершается
- Можно играть одному или вдвоем на одном компьютере

## Управление

| Режим | Игрок 1 | Игрок 2 |
|:-----:|:-------:|:-------:|
| **Одиночный** | ← ↑ → ↓ | — |
| **Для двоих** | ← ↑ → ↓ | **W A S D** |

**Дополнительно:**
- Кнопка **"Назад"** в игре — возврат в главное меню
- Кнопки **включения/выключения звука** в меню
- **Любая клавиша** — начало игры после заставки

## Установка и запуск

### Вариант 1: Запуск готовой сборки
1. Скачайте последний релиз из раздела [Releases](https://github.com/a1inas/Snake-Game-WPF/releases) (когда он появится)
2. Распакуйте архив
3. Запустите `Snake.exe`

### Вариант 2: Сборка из исходного кода

```bash
# Клонируйте репозиторий
git clone https://github.com/a1inas/Snake-Game-WPF.git

# Перейдите в папку проекта
cd Snake-Game-WPF
cd Snake  # Все исходные файлы находятся в этой папке
```

**Требования для сборки:**
- Visual Studio 2022
- .NET 6.0 SDK
- Платформа: Windows

**Шаги:**
1. Откройте файл решения `Snake.sln` в Visual Studio
2. Выберите конфигурацию **Release**
3. Нажмите **Build → Build Solution** (или `Ctrl+Shift+B`)
4. Запустите собранный `.exe` файл из папки `bin/Release/net6.0-windows/`

## Ключевые механики

### Игровое поле
Поле размером **25×25** клеток, где каждая клетка может быть пустой, содержать змейку или еду. Змейка не может выйти за границы поля.

### Движение змейки
Змейка движется непрерывно в текущем направлении. Игрок может менять направление с помощью клавиш управления. Нельзя развернуться в противоположном направлении.

### Поедание еды
При соприкосновении головы змейки с едой:
- Змейка увеличивается на один сегмент
- Счет игрока увеличивается на 1
- На поле появляется новая еда в случайном месте

### Столкновения
Игра завершается если:
- Голова змейки касается стены
- Голова змейки касается собственного тела
- В режиме для двоих: голова одной змейки касается тела другой

### Анимация смерти
При проигрыше змейка "замирает" и меняет цвет, после чего появляется окно с результатами.
